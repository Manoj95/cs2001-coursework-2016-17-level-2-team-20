Brunel Planner

Brunel Planner is an app that allows students to keep track of their current timetable in a mobile friendly format. The app allows users to customise features to add specific events alongside saving any relevant details such as lecture notes.

Timetable

The Timetable page allows a user to view their individual timetable. The day of the week is displayed alongside the name of the module, lecturer, time of lecture and where the lecture is going to be held.

Events

The Events page allows a user to create new custom events and assign them to a specific date and modules.

Notes

The Notes page allows a user to create new custom notes and assign them to specific modules.

Maps

The Maps page allows a user to find their way around Brunel buildings.
